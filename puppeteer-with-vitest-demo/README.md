# Puppeteer with Vitest Demo

This is a demo to show how to use Puppeteer with Vitest to test AI actions and assertions.

create `.env` file

```shell
# replace by your own
OPENAI_API_KEY="YOUR_TOKEN"
```

run demo

```bash
npm install 

npm run test
```
